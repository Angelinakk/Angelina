Uvod

JavaScript je interpretiran objektno orijentisan skriptovan jezik visokog nivoa.
On se dominantno koristi pri izradi web stranica uz HTML i CSS
JavaScript sadrzi biblioteku objekata kao sto su:

  • Niz
  • Datum
  • Matematika
  • Operatori
  • Kontrolne strukture
  • Iskazi

DOM omogucava strukturni prikaz dokumenta i definise nacin kako se moze pristupiti
iz programa i kako oni mogu promeniti njegov sadrzaj, stil i strukturu.
Koriscenjem Document Object modela mogu da se unesu brojne promene na samoj stranici:

  • menjanje HTML elemenata;
  • promena CSS stilova;
  • uklanjanjanje HTML elemenata i atributa;
  • dodavanje novih HTML elemenata i atributa;
  • upravljanje dogadjajima;
  • kreiranje novih HTML dogadjaja.

Sta vec treba da znate

Da bi nastavili sa citanjem i primenjivanjem sadrzaja ovog sajta neophodno je da imate znanje iz:

  • Opsteg poznavanja interneta
  • Dobro poznavanja HTMLa
  • Malo iskustva u programiranju

Hello world

Zapocinjemo sa pisanjem Hello world tako sto cemo ispisati sledeci kod:
function greetMe(yourName) { alert("Hello " + yourName); }
greetMe("World");

Promenljive/varijable

Koristimo varijable kao simbolicna imena koja nam pomazu da pristupimo sadrzaju te varijable. Pravila vezana za nazive varijabla JavaScript-a su:

  • nazivi varijabli su osetljivi na veliko i malo slovo (s i S su dve drugačije varijable);
  • ime varijable može počinjati i znakom $ ili _
  • znakovi mogu da budu i cifre

Deklarisanje varijabli

Varijablu mozete deklarisati na tri nacina:

  • Uz rec var koja vazi u celom fajlu
    var x = 42.
  • Jednostavnim dodeljivanjem vrednosti к = 42.
  • Uz kljucnu rec let koja vazi samo u bloku u kom je deklarisana
    let y = 13.

Opsek varijabli

Deklarisana varijabla izvan funkcije zove se globalna promenljiva, jer de dostupna u celom dokumentu.
Deklarisana varijabla unutar funkcije zove se lokalna promenljiva, jer je dostupna samo unutar te funkcije.

Globalne varijable

Globalna varijabla je vrsta varijable koja je deklarirana izvan bilo koje funkcije
i dostupna je svim funkcijama u programu.
Globalni objekat je window tako da mozemo pristupiti globalnim promenljivim
koristeci window.variable syntax.

Konstante

Koristite const za deklarisanje varijabli ako možete predvideti da ta varijabla
neće dobijati drugu vrednost. Ako imate razloga da mislite
da će nekad tokom njenog korištenja ta varijabla dobiti drugu vrednost
koristite let ključnu reč.
const PI = 3.14;
*Ne mozete deklarisati konstantu sa istim imenom kao funkcija ili promenljivu
tom delu, to ne vazi za objekte.

Tipovi podataka

JavaScript ima sledeće tipove podataka:

  • logička ili Bulova vrednost (boolean), može biti tačno (true) ili netačno (false)
  • null, koristi se da predstavi nepostojeću vrednost
  • undefined, za vrednost promenljivih koje su deklarisane ali im još nije dodeljena
    vrednost (i za povratnu vrednost funkcija koje ne vraćaju vrednost)
  • broj (number), može biti ceo ili decimalni
  • string sadrži znakove u navodnicima, npr. “Zdravo Svete”
  • simbol
  • object

if...else

Koristite if naredbu da odaberete blok JavaScript koda koji ce se izvrsiti ako je uslov tacan.
Mozete da koristite if else da biste testirali vise uslova u nizu.
U slucaju visestrukih uslova bice izvrsen samoprvi logicki uslov koji je tacan

while

While izraz se izvršava sve dok je uslov TRUE, a ukoliko nije onda se izlazi iz petlje.
Ovaj tip petlje se najčešće koristi kada se ne zna tačan broj izvšenja petlje.
U slučaju da uslov na samom početku nije ispunjen, petlja se ne izvršava ni jednom.
Pre ulaska u petlju vrši se inicijalizacija promenljivih koje će biti korišćene bilo u uslovu,
bilo u telu petlje. U telu petlje se nalazu kod koji formira uslov za ostanak u petlji.

Deklaracija funkcija

Deklaracija funkcija sastoji se od kljucne reci funkcije:

  • naziv funkcije
  • lista argumenata funkcije
  • JavaScript izjave koje definisu funkciju

Primer:
function square(number) { return number * number; }
Funkcija kvadrat uzima argument koji se zove broj, funkcija se sastoji od jedne
izjave koja kaze da vrati argument funkcije pomnozen sam sa sobom.
